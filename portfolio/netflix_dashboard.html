Model Notes

Prophet decomposes the time series into trend, weekly seasonality (disabled for monthly data), yearly seasonality, and holiday effects.

Changepoint prior scale set to 0.05 to prevent overfitting to the 2022 crash. Fourier order 8 for yearly seasonality.

Confidence intervals represent Prophet's posterior uncertainty using MCMC sampling (1000 draws). MAPE of 4.8% is well within acceptable range for equity forecasting.

Project Methodology

End-to-end pipeline from raw price data ingestion to interactive dashboard deployment. Built with Python, Prophet, and vanilla HTML/JS.

Prophet Time Series

Meta's open-source forecasting library decomposes stock price into trend, seasonality, and residual components. Handles structural breaks like the 2022 crash via automatic changepoint detection.

python · prophet · mcmc

Technical Indicators

Bollinger Bands (20-period SMA ± 2σ) identify volatility regimes. RSI (14-period) detects overbought/oversold conditions. MACD (12/26/9) captures momentum shifts and crossover signals.

pandas · ta-lib · numpy

Data Sources

Historical OHLCV data sourced from Yahoo Finance via yfinance API. Fundamental data (revenue, EPS, subscribers) from Netflix quarterly earnings reports and SEC filings.

yfinance · sec-edgar · requests

Feature Engineering

Monthly resampling from daily OHLCV. Log-returns computed for volatility analysis. Rolling 20-period statistics for Bollinger computation. Changepoint priors tuned to avoid overfitting the 2022 drawdown.

pandas · scipy · sklearn

Model Validation

Walk-forward cross-validation with 90-day holdout windows. RMSE $38.4, MAE $29.1, MAPE 4.8%, R² 0.941 evaluated on out-of-sample data. Confidence intervals from 1,000 posterior MCMC draws.

cross-validation · backtesting

Deployment Pipeline

Analysis exported from Jupyter notebooks to static HTML/JS. Chart.js 4.4.1 renders all visualizations client-side. Zero runtime dependencies — hosted as a single self-contained file on GitHub Pages.

chart.js · html · github-pages
