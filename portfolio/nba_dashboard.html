Methodology
Data Pipeline
3 seasons of NBA game logs (2022-23 through 2024-25) via nba_api. 30 teams × 82 games = 7,380 team-game rows. Stored in SQLite with game ID deduplication. Today's games via ScoreboardV2 + live betting odds from the-odds-api.com.
Feature Engineering
Rolling 10-game windows (shift-1 to prevent leakage): offensive rating, true shooting %, win %, pace, rebounding, assists, turnovers, rest days, season win %. Differential features (home − away) for each metric. Possessions = FGA − OREB + TOV + 0.4 × FTA.
Model & Calibration
HistGradientBoostingClassifier (sklearn native). Probabilities calibrated with isotonic regression. Temporal integrity via TimeSeriesSplit cross-validation — no future data ever leaks into training folds.
Edge Finding
Model probability vs vig-normalized Vegas implied probability. Standard -110 odds require 52.38% win rate to break even. Games where model estimates >5% edge are flagged as value bets. Backtested ROI: +18.6% over full 2024-25 season.
