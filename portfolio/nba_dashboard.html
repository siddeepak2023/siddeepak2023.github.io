Methodology

Data — 3 seasons of NBA game logs (2022-23 through 2024-25) via nba_api. 7,380 team-game rows → 3,643 paired matchup rows.

Features — Rolling 10-game windows (shift-1, no leakage): offensive rating, true shooting %, win %, pace, rebounding, assists, turnovers, rest days, season win %. Differential (home minus away) for each metric.

Model — HistGradientBoostingClassifier trained on 2022-23 + 2023-24, evaluated on 2024-25 holdout. Probabilities calibrated with isotonic regression (5-fold CV). Temporal integrity via TimeSeriesSplit.

Edge finding — Model probability vs vig-normalized Vegas implied probability. Games with >5% edge flagged as value bets.
